# Financial-fraud-analytics-dashboard
A machine learning-powered fintech analytics system designed to detect fraudulent financial transactions and provide real-time risk insights through an interactive Streamlit dashboard.

This project utilizes the PaySim mobile money transaction dataset to simulate real-world banking fraud scenarios and applies a Random Forest classifier for fraud prediction and transaction risk analysis. The system analyzes transaction behavior based on transaction type, amount, temporal activity, and fraud indicators to identify suspicious patterns and potential financial threats.

The dashboard provides:

real-time fraud prediction
fraud probability scoring
transaction analytics
fraud distribution visualization
model performance metrics
confusion matrix and evaluation reports

Built with a modern fintech-inspired UI, the platform mimics real-world banking transaction monitoring systems used in fraud detection, anti-money laundering (AML), and financial risk analytics environments.
